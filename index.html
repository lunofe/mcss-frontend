Recommended

Here you can find the recommended softwares for your use case. This list is based on our opinions and might not be the golden standard.

Note! Some versions are outdated and shouldn't be used. There is, for example, no reason to use versions like 1.13 or 1.17, as newer versions are running far better.

üåç Minecraft Vanilla

  • If you just want to play Minecraft without any modifications, you can use the Vanilla server.
  • If you want to have some bug fixes, settings and better performance, you can use Fabric with the Carpet or Lithium mod.

üö∞ Minecraft Vanilla with plugins¬°

  • If you want to use plugins on your 1.19+ server, use Paper or Pufferfish or Purpur.
  • If you want to use plugins on your 1.8-1.18 server, use Paper Legacy.
  • If you want to use plugins on your 1.7.10 server, use PaperSpigot.
  • If you want to use plugins on Beta 1.7.3 use Canyon.

üî® Minecraft with mods

  • If you want to use mods on your server, you can use Forge or Fabric, depending on which mods you want to use.

üî∂ Minecraft with mods and plugins

  • If you want to use Forge mods and plugins on 1.20.2-1.20.4 you can use Ketting (This is a development version).
  • If you want to use Forge mods and plugins on 1.19.2/1.19.4 you can use Arclight.
  • If you want to use Forge mods and plugins on 1.18.2 you can use Magma Maintained
  • If you want to use Forge mods and plugins on 1.16.5 you can use Mohist or Arclight.
  • If you want to use Forge mods and plugins on 1.12.2 you can use CatServer.
  • If you want to use Forge mods and plugins on 1.7.10 you can use Crucible.
  • If you want to use Fabric mods and plugins, you can use Mohist Banner or Cardboard.

üìô Minecraft Bedrock

  • If you want to make a Minecraft Bedrock Server with plugins, use Spigot or Paper in combination with Geyser.

üåê Minecraft Networks

  • If you want to connect Java Edition servers with each other, use Velocity.
  • If you want to connect Bedrock Edition servers with each other, use WaterdogPE.
